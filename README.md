# MindMooc - 慕课视频思维导图生成器

🎓 **基于Vue3的智能视频思维导图生成平台**

一个现代化的Web应用，能够将MOOC视频自动转换为结构化的思维导图，帮助用户更好地理解和记忆学习内容。

## ✨ 项目特色

### 🎯 核心功能

- **智能转换**: 上传视频自动生成Mermaid格式思维导图
- **批量处理**: 支持多个视频文件同时处理
- **可视化编辑**: 内置思维导图编辑器，支持自定义修改
- **历史管理**: 完整的任务历史记录和管理功能
- **用户系统**: 个人中心、反馈系统等完整用户体验

### 🎨 界面设计

- **现代化UI**: 采用Element Plus组件库，界面简洁美观
- **绿色主题**: 与品牌logo完美匹配的渐变色彩方案
- **响应式布局**: 完美适配桌面、平板、移动端设备
- **流畅交互**: 丰富的悬停效果、过渡动画和视觉反馈

### 🚀 创新亮点

- **可拖拽分割布局**: 任务详情页支持自由调整视频和思维导图显示比例
- **双向尺寸调整**: 支持宽度和高度的独立调整，最大化利用屏幕空间
- **智能高度限制**: 自动适应屏幕尺寸，最高可达屏幕高度的80%

## 📋 页面架构

### 主要页面

1. **🏠 首页 (Home)**: 产品介绍和功能展示，引导用户开始使用
2. **📝 新建任务 (NewTask)**: 单个视频文件上传和处理
3. **📚 批量任务 (BatchTask)**: 多个视频文件批量处理，支持任务命名
4. **📋 历史任务 (HistoryTask)**: 查看所有历史任务，支持详情查看和删除
5. **👤 个人中心 (Profile)**: 用户信息管理、密码修改、使用统计

### 辅助页面

1. **🔍 任务详情 (TaskDetail)**: 视频播放与思维导图对照查看
2. **✏️ 导图编辑 (MapEditor)**: 思维导图自定义编辑功能
3. **💬 反馈页面 (Feedback)**: 用户意见和建议收集

## 🛠️ 技术栈

### 前端框架

- **Vue 3**: 采用Composition API的现代化前端框架
- **Vite**: 快速的构建工具和开发服务器
- **Vue Router 4**: 单页应用路由管理

### UI组件库

- **Element Plus**: 专业的Vue 3组件库
- **@element-plus/icons-vue**: 丰富的图标库

### 核心依赖

- **Mermaid**: 强大的图表和流程图渲染引擎
- **Axios**: HTTP客户端，用于API通信

## 🎮 核心功能详解

### 📝 任务创建

- **拖拽上传**: 支持拖拽和点击两种上传方式
- **格式验证**: 自动验证视频文件格式和大小
- **实时反馈**: 上传进度和状态实时显示

### 📚 批量处理

- **任务命名**: 为批量任务设置自定义名称
- **进度跟踪**: 每个文件的处理进度独立显示
- **状态管理**: 待处理、处理中、已完成、失败等状态管理

### 🔍 智能查看界面

- **并排布局**: 视频和思维导图并排显示，便于对照学习
- **可拖拽调整**: 
  - 水平拖拽调整宽度比例 (20%-80%)
  - 垂直拖拽调整整体高度 (最小500px，最大屏幕高度80%)
- **响应式适配**: 移动端自动切换为垂直堆叠布局

### 👤 用户管理

- **个人信息**: 用户名、邮箱等基本信息管理
- **密码安全**: 支持密码修改，包含确认验证
- **使用统计**: 总任务数、完成数、处理中、失败任务等统计
- **反馈系统**: 功能建议、问题反馈、使用咨询等分类反馈

## 📱 响应式设计

### 桌面端 (>1024px)

- 完整功能体验
- 支持所有拖拽调整功能
- 侧边栏导航布局

### 平板端 (≤1024px)

- 垂直堆叠布局
- 简化的交互方式
- 保持核心功能完整性

### 移动端 (≤768px)

- 优化的触摸交互
- 紧凑的界面布局
- 关键功能保留
